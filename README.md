### 📊 Project Overview
This project is a high-fidelity replication of a professional payment orchestration and market intelligence report, styled after official S&P Global financial publications. The goal of this dashboard is to transform complex financial transaction and market data into a structured, executive-level visual narrative that supports strategic decision-making.

---

### 🛠️ Key Features & Technical Implementation
* *Advanced Data Modeling:* Structured around a highly optimized Star Schema to ensure efficient cross-filtering, quick load times, and data integrity.
* *Complex DAX Calculations:* Implemented advanced time-intelligence metrics, performance tracking, and specific financial growth formulas (such as YoY growth and custom period comparisons).
* *Executive-Level UI/UX:* Designed with a clean, corporate-aligned color palette, intuitive navigation, and precise visual alignment to match institutional reporting standards.

---

### 🔧 Tech Stack Used
* *BI Platform:* Microsoft Power BI Desktop
* *Data Transformation:* Power Query (ETL process, data cleaning, and type formatting)
* *Analytical Calculations:* DAX (Data Analysis Expressions)
